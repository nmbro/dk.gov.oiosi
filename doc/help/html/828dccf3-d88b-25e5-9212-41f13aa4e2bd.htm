IChannelInterceptor Interface
Members  See Also  Send Feedback
Interface that describes the methods an interceptor on a reply channel should implement. There are two methods "InterceptRequest" and "InterceptResponse" that needs to be implemented. The first is for incoming communication and the second is for outgoing communication.
